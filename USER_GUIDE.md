# دليل استخدام The Sphere — للمستخدم والموظف والأدمن

> الموقع: **https://thesphere-eight.vercel.app**

التطبيق فيه 3 أنواع استخدام:
- **الزائر / ولي الأمر** (مايحتاجش تسجيل دخول)
- **الموظف (Staff)** — مكتب الاستقبال: تشييك إن/آوت وتسجيل الأنشطة
- **الأدمن** — لوحة تحكم كاملة لإدارة المركز

---

## أولاً: الزائر / ولي الأمر 👨‍👩‍👧

ده الجزء العام — أي حد يقدر يدخله من غير حساب.

| الصفحة | الرابط | بتعمل إيه |
|--------|--------|-----------|
| الرئيسية | `/` | بوابة The Sphere + الـ 11 land |
| الأراضي | `/lands` | كل الأنشطة، وكل land جواه المحطات |
| المتصدّرين | `/leaderboard` | ترتيب الأطفال بالنقاط (بالاسم المختصر فقط) |
| البروفايل | `/profile/{id}` | نقاط الطفل، شاراته، وأكواد الخصم |
| "دوّر على بروفايلي" | `/login` | إدخال رقم تليفون ولي الأمر لاسترجاع بروفايل الطفل |

**إزاي تبدأ كزائر:**
1. الطفل بيتسجّل عند مكتب الاستقبال (الموظف بيعمل ده) وبياخد **سوار** (NFC) و/أو **QR code**.
2. بعد التسجيل بتظهر صفحة البروفايل تلقائياً — احفظ الرابط أو امسح الـ QR.
3. كل زيارة الطفل بيكسب **نقاط** حسب الوقت والأراضي اللي زارها، و**شارات** عند الإنجازات، و**أكواد خصم** للزيارات المتكررة.
4. تقدر تشوف ترتيب طفلك في `/leaderboard` (بنعرض اسم مختصر فقط للخصوصية).

> 🔒 خصوصية: بياناتك (اسم ولي الأمر/التليفون الكامل) **مش بتظهر** لأي حد على النت — بس النقاط والاسم المختصر.

---

## ثانياً: الموظف (Staff) 🎫

شغل مكتب الاستقبال: تسجيل الأطفال، الدخول والخروج، وتسجيل الأنشطة.

**تسجيل الدخول:** `/staff/login` بالإيميل والباسوورد اللي الأدمن أداهولك.
بعد الدخول بتتحوّل لصفحة **`/checkin`**.

### تسجيل طفل جديد — `/register`
1. املأ: اسم الطفل، السن (4–18)، اسم ولي الأمر، رقم التليفون.
2. اضغط **Register** → التطبيق بيكتب السوار (NFC) تلقائياً لو الجهاز بيدعم.
3. لو مفيش NFC، بيظهر **QR code** — اطبعه أو صوّره وحطه مع السوار.

### تشييك إن (دخول) — `/checkin` → تبويب **Check In**
1. امسح سوار/QR الطفل (أو NFC).
2. بيظهر بروفايل الطفل → اضغط **Confirm Check In** → بتبدأ الجلسة (الوقت بيتحسب).

### تسجيل الأنشطة (اختياري)
ممكن تسجّل دخول/خروج الطفل من كل land عشان الحساب يكون دقيق.

### تشييك آوت (خروج) — تبويب **Check Out**
1. امسح سوار/QR الطفل.
2. اختار الأراضي اللي زارها والساعات.
3. اضغط **Check Out & Calculate Points** → التطبيق بيحسب:
   - **النقاط** (بالساعات + البونصات + مضاعِف الـ streak)
   - **الشارات** الجديدة
   - **كود خصم** لو استحقّه
4. بتظهر شاشة احتفال بالنتيجة.

> ✅ أمان: الخروج **مايتكررش** — لو اتعمل مرتين بالغلط، المرة التانية مش بتزوّد نقاط. والساعات بتتحسب من السيرفر فمحدش يقدر يزوّدها.

## ثالثاً: الأدمن 🛠️

لوحة تحكم كاملة على `/admin` (لازم تسجّل دخول بحساب صلاحيته **admin**).

| القسم | الرابط | بيعمل إيه |
|-------|--------|-----------|
| لوحة القيادة | `/admin` | نظرة عامة |
| الأراضي | `/admin/lands` | تعديل الـ lands (الاسم، اللون، الوصف، الفتح/الغلق) والمحطات |
| الجدول | `/admin/schedule` | تحديد الأراضي/المحطات المفتوحة في كل يوم |
| الكامبرز | `/admin/campers` | البحث عن الأطفال وبياناتهم |
| إعدادات النقاط | `/admin/points-config` | تعديل قيم النقاط والبونصات والمضاعِفات |
| إعدادات الخصومات | `/admin/discount-config` | نسب الخصم (ولاء، مجموعة، إخوات، حفلات...) |
| الحفلات | `/admin/ceremonies` | إنشاء حفل يومي + تسجيل الفائزين وتوليد أكواد جوائزهم |
| صور الهوم | `/admin/home-photos` | تغيير صور الصفحة الرئيسية |

### إنشاء موظف أو أدمن جديد
الحسابات بتتعمل على مرحلتين (مرة واحدة لكل موظف):
1. **Supabase Dashboard → Authentication → Users → Add user**: اعمل إيميل + باسوورد للموظف.
2. انسخ الـ User UID، وفي **SQL Editor** نفّذ:
   ```sql
   insert into public.staff_users (user_id, role)
   values ('USER-UID-HERE', 'staff');   -- أو 'admin' للأدمن
   ```
3. الموظف يقدر يدخل من `/staff/login`. الـ `staff` بيوصل `/checkin` بس، والـ `admin` بيوصل `/admin` كمان.

> لتغيير صلاحية موظف لأدمن: `update public.staff_users set role='admin' where user_id='...';`
> لإلغاء وصول موظف: `delete from public.staff_users where user_id='...';` (ويفضّل تعطيل حسابه في Authentication كمان).

## ملاحظات تشغيلية مهمة

- **NFC** بيشتغل على **Android + Chrome** فقط (أجهزة الموظفين). على iPhone أو الكمبيوتر استخدم **QR code** بدلاً منه — التطبيق بيتحوّل له تلقائياً.
- **الأساور مؤمّنة بتوقيع رقمي** — أي سوار مزوّر أو معدّل بيترفض عند المسح.
- **النقاط/الخصومات قابلة للتعديل** من لوحة الأدمن من غير ما تلمس الكود.
- **النسخ الاحتياطي**: فعّل Point-in-Time Recovery من Supabase لحماية بيانات الأطفال.

## استكشاف الأخطاء السريع
| المشكلة | الحل |
|---------|------|
| "Not authenticated" عند فتح `/admin` أو `/checkin` | سجّل دخول من `/staff/login` |
| "This account does not have staff access" | الحساب موجود في Auth بس مش متضاف في `staff_users` — أضِفه (فوق) |
| السوار مش بيتقري | استخدم QR، أو اتأكد إن الجهاز Android/Chrome وإن NFC مفعّل |
| "Invalid wristband signature" | السوار مش صادر من النظام أو اتعدّل — أعد تسجيل الطفل |
| الموقع بيظهر بيانات فاضية | اتأكد إن متغيرات البيئة مظبوطة في Vercel (Supabase keys) |
